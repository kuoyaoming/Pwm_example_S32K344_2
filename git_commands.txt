# Git 命令执行顺序
# 安装 Git 后在此目录运行以下命令：

# 1. 添加所有更改
git add .

# 2. 提交更改
git commit -m "清理 gitignore 文件：移除构建产物和自动生成的文件

- 删除 Debug_FLASH/ 目录（包含所有编译产物）
- 删除 generate/ 目录（自动生成的代码）
- 删除配置文件：ClockConfigurationMappings.txt, ClockYaml.txt
- 清理 *.o, *.d, *.elf, *.map, *.args 文件
- 清理 makefile 相关文件

这些文件应该被 .gitignore 忽略，不应提交到版本控制中。"

# 3. 推送到 GitHub
git push

# 4. 检查状态
git status
